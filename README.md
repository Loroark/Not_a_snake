# Not_a_snake
Game like snake but not a snake.


This game might look like snake but it's way worse XD

Overall

In this game you steer blue dot, collect green dot and avoid red dot. That's all :)

Steer with arrow keys
Press '1' for easy
Press '2' for hard
Press 'P' for Polish
Press 'E' for English
Press 'R' to restart
Press 'ESC' to exit
